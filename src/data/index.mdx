# Install Virtool

## Docker <Recommended />

1. Write a `docker-compose.yml` similar to the following.

   This will run Virtool and MongoDB in containers and persist their data in volumes.

   ```yaml
   version: "3.1"
   services:
      mongo:
        image: mongo:3.6
         ports:
            - "27017:27017"
         volumes:
            - mongo:/data/db
      virtool:
         image: "virtool:legacy-1"
         environment:
            VT_DATA_PATH: "/data"
            VT_HOST: "0.0.0.0"
         ports:
            - "9950:9950"
         volumes:
            - virtool:/data
   ```

## Binary

The legacy method for installing Virtool is to

1. Download the latest Virtool release from the [Virtool website](http://www.virtool.ca "Virtool Website") and unpack the downloaded archive

   ```shell
   tar -xvf virtool.tar.gz
   ```

2. Move the unpacked directory to your installation location

   ```shell
   mv virtool /opt
   ```

3. Start Virtool by executing the `run` file

   ```shell
   ./run
   ```
